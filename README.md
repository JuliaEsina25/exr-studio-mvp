# EXR Studio - Интернет-магазин одежды

## 🛍️ О проекте
Интернет-магазин одежды с возможностью заказа по индивидуальным меркам. 
Проект разработан как MVP (Minimum Viable Product) для демонстрации функционала.

## ✨ Функционал
- Каталог товаров с фильтрацией
- Карточки товаров с галереей изображений
- Корзина покупок
- Оформление заказа
- Авторизация и регистрация
- Расчет ткани для пошива
- Личный кабинет пользователя
- Отслеживание заказов

## 🛠️ Технологии
- PHP 7.4+
- MySQL
- HTML5/CSS3
- JavaScript (чистый, без фреймворков)
- Bootstrap 5
